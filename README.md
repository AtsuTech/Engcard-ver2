## 技術
laravelをベースで使用し、inertia.jsでフロントエンドをReactで表示するSPA仕様にしている


- **laravel(ver.12)**
- **[inertia(laravelコントローラの値をReactに渡すライブラリ)](https://inertiajs.com/)**
- **react**
- **react-chartjs-2**


## 機能
- CRUD処理
- ファイル(画像)アップロード
- 認証機能



## 本番環境
- サーバー: Xserver VPS
- webサーバ: apache
- データベース: mysql
